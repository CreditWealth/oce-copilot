# One Credit Engine Plugin

This plugin folder contains custom GitHub Copilot assets for the One Credit Engine (OCE) workspace.

## Overview

The plugin is used to organize OCE-specific agent definitions and related customization files.

## Contents

- `agents/` contains custom agents for documentation and other OCE-focused workflows.
- `agents/doc-writer.agent.md` defines the **OCE Doc Writer** agent for Markdown documentation work.
